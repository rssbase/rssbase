# RssBase

## Requirements

- [PHP](https://www.php.net/downloads.php) `>= 8.1` & [composer](https://getcomposer.org/download/) `>= 2.0`

## Development

```
$ make

# Project
install                        Install project dependencies
clean                          Remove generated files

# Utils
cs                             Run php-cs-fixer
rector                         Run rector

# Tests
tests                          Run all tests
tu                             Run unit tests
tf                             Run functional tests
```
